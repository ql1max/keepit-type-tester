# Typography Testing Tool

An interactive tool for fine-tuning letter spacing and testing typography with Plus Jakarta Sans.

## Features

- Real-time letter spacing adjustment with live preview
- Side-by-side comparison view
- Custom text testing
- Dark mode support
- Preset typography scales
- CSS export functionality

## Usage

Visit the live tool and use the sliders to adjust letter spacing across different text sizes. Test your adjustments with custom text, toggle between light and dark modes, and export your CSS values when ready.

## Local Development

Open `index.html` in your browser, or use a local server:

```bash
# Python
python -m http.server 8000

# Node.js
npx http-server
```
